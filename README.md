ws-proxy
========================================================================================================================
ws-proxy is just a proxy for websocket protocol.
This is similar to wscat (nodejs app), but can only use as websocket client.

How to Install
------------------------------------------------------------------------------------------------------------------------
```
go get -u -v github.com/kamichidu/ws-proxy
```

How to Use
------------------------------------------------------------------------------------------------------------------------
```
$ ws-proxy -h
$ ws-proxy -v
$ ws-proxy -u ws://localhost
```
